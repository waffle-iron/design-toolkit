Basic Tables

Use tables sparingly, only for tabular data. Use the nypl-basic-table class to format a table. All tables should have a thead element with the corresponding headers.

Example

some value some value some value some value
some value some value some value some value
… … … …

Search results summary

Presents a comma-separated list of search terms or filters along with the number of results. The result count has a special span tag (should be number-formatted e.g.: 1,232). Each applied facet or term has a button element to remove the facet and refresh the search results. the button contains the screen-reader-friendly text inside a span tag.

Accessibility note: The main element must include the following attributes when being updated via JavaScript (no page refresh): aria-live="assertive" aria-atomic="true" role="presentation".

NOTE: Avoid adding a space between the facet and the remove button so that it won't wrap to the next line.

Example
1,807 resources found for Young Adult , Fiction . 

.nypl-results-summary
.nypl-results-count
.nypl-facet
    

<div class="nypl-results-summary" aria-live="assertive" aria-atomic="true" role="presentation">
  <span class="nypl-results-count">1,807</span> resources found for
  <span class="nypl-facet">
    <strong>Young Adult<button><span>Remove Young Adult filter</span></button></strong>
  </span>,
  <span class="nypl-facet">
    <strong>Fiction<button><span>Remove Fiction filter</span></button></strong>
  </span>.
</div>

Sorting

Sorters are nested inside a nypl-results-sorting-controls element. The sorters themselves implement a nypl-results-sorter microformat which is based on the Navigation Menu Button and use the aria-expanded attribute that must be modified accordingly by the appropriate JavaScript (see the Navigation Menu Button documentation for more details). The sorting options are nested in an unordered list with a menu role.

Example
  • relevance
  • title A-Z
  • title Z-A
  • date (old to new)
  • date (new to old)

.nypl-results-sorting-controls
.nypl-results-sorter
    

<div class="nypl-results-sorting-controls">
  <div class="nypl-results-sorter">
    <button aria-expanded="false">
    Sort by <strong>relevance</strong>
      <svg aria-hidden="true" class="nypl-icon" preserveAspectRatio="xMidYMid meet" viewBox="0 0 68 24">
        <title>wedge down</title>
        <polygon points="67.938 0 34 24 0 0 10 0 34.1 16.4 58.144 0 67.938 0" />
      </svg>
    </button>
    <ul class="hidden" role="menu">
      <li><a href="?sort=relevance"><strong>relevance</strong></a></li>
      <li><a href="#">title A-Z</a></li>
      <li><a href="#">title Z-A</a></li>
      <li><a href="#">date (old to new)</a></li>
      <li><a href="#">date (new to old)</a></li>
    </ul>
  </div>
</div>

Search results (non-table)

Sometimes search results don't accommodate to a consistent structure or it is not useful to present them in table form. In these cases results should be returned in a unordered list (ul) of class nypl-results-list. Each result list item should have the class nypl-results-item and contain:

  • An h2 element that contains a link to the result page.
  • Optional: A notification of unavailability of the item (if applicable) in a div with class nypl-results-unavailable.
  • Optional: A definition list (dl) with the most relevant attributes of the result item sorted by importance. Terms that need to be highlighted (for easier scanning of results) should have a class of nypl-results-highlight

Example

  • The Biography of Example

    Material Type
    Text
    Author
    Smith, Angela F., 1982-
    Published
    2014

  • Very Rare Item

    Currently not available for use
    Material Type
    Text
    Author
    Casasbuenas Gutiérrez, Margarita, 1915-1997
    Published
    1942

.nypl-results-list
.nypl-results-item
.nypl-results-unavailable
.nypl-results-highlight
    

<ul class="nypl-results-list">
  <!-- an item -->
  <li class="nypl-results-item">
    <h2><a href="#">The Biography of Example</a></h2>
    <dl>
      <dt class="nypl-results-highlight">Material Type</dt>
      <dd class="nypl-results-highlight">Text</dd>
      <dt>Author</dt>
      <dd>Smith, Angela F., 1982-</dd>
      <dt>Published</dt>
      <dd>2014</dd>
    </dl>
  </li>
  <!-- another item -->
  <li class="nypl-results-item">
    <h2><a href="#">Very Rare Item</a></h2>
    <div class="nypl-results-unavailable">Currently not available for use</div>
    <dl>
      <dt class="nypl-results-highlight">Material Type</dt>
      <dd class="nypl-results-highlight">Text</dd>
      <dt>Author</dt>
      <dd>Casasbuenas Gutiérrez, Margarita, 1915-1997</dd>
      <dt>Published</dt>
      <dd>1942</dd>
    </dl>
  </li>
</ul>

Search results pagination

This widget is presented only in the bottom of the results (pagination will never appear on top). Page count appears in its own span tag.

Example
