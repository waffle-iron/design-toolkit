NYPL Logo

The NYPL branding / logos have special use cases. You'll probably have no need to modify the logo other than changing its size. We can use either the logo mark or the logo type. This can be achieved by using css to hide either of these two components of the logo mark.

Example {% include rawSvg/logoLion.icon.svg %} 

    .nypl-icon {
    @include nypl-icon($height: 10em, $width: 10em);
    }
